Användarvillkor

Här kan du läsa vilka villkor som gäller när du använder Weck Technologies AI-baserade tjänster.

Användarvillkor
Weck Technologies

1. Allmänt
Dessa användarvillkor gäller för användning av Weck Technologies AI-baserade tjänster.
Tjänsten riktar sig endast till företag (B2B). Genom att använda tjänsten accepterar kunden dessa villkor.

2. Tjänsten
Weck Technologies tillhandahåller en AI-baserad receptionist/chattlösning som besvarar frågor och guidar användare.
Tjänsten är ett hjälpmedel och innebär ingen garanti för affärsresultat, leads eller försäljning.

3. Gratis testperiod
Kunden erbjuds 14 dagars kostnadsfri testperiod.

Under testperioden kan tjänsten användas fullt ut.

4. Abonnemang och betalning
Efter testperioden övergår tjänsten automatiskt till ett abonnemang.

Pris: 2 995 kr per månad exkl. moms.

Fakturering sker månadsvis via bokförings-/faktureringstjänst (t.ex. Bokio).

Betalningsvillkor: 14 dagar.

Ingen bindningstid. Kunden kan avsluta när som helst.

Vid utebliven betalning har Weck Technologies rätt att pausa eller avsluta tjänsten.

5. Kundens ansvar
Kunden ansvarar för:

Innehåll som tillhandahålls till tjänsten
Att nödvändiga rättigheter finns för den data som behandlas
Att informera sina slutkunder om att AI-baserad kommunikation används

6. Drift och tillgänglighet
Weck Technologies strävar efter hög tillgänglighet men garanterar inte oavbruten drift.
Tillfälliga avbrott kan förekomma vid underhåll eller tekniska problem.

7. Ansvarsbegränsning
Weck Technologies ansvarar inte för indirekta skador, utebliven vinst eller affärsförluster.
Eventuellt skadeståndsansvar är begränsat till det belopp kunden betalat för tjänsten under den senaste månaden.

8. Uppsägning
Kunden kan avsluta tjänsten när som helst.
Weck Technologies har rätt att avsluta tjänsten vid brott mot villkoren.

9. Personuppgiftsbiträdesavtal (DPA)
9.1 Roller
Kunden är personuppgiftsansvarig.

Weck Technologies är personuppgiftsbiträde.

9.2 Behandling av personuppgifter
Weck Technologies behandlar personuppgifter endast för att tillhandahålla tjänsten och enligt kundens instruktioner.
Personuppgifter kan avse både kundens representanter och kundens slutkunder som interagerar via chatt.

9.3 Säkerhet
Weck Technologies vidtar rimliga tekniska och organisatoriska åtgärder för att skydda personuppgifter.

9.4 Underbiträden
Personuppgifter kan behandlas av underbiträden (t.ex. AI-leverantörer, hosting, bokföring) i den utsträckning som krävs för tjänstens funktion.

9.5 Avslut
Vid avslutat avtal raderas eller anonymiseras personuppgifter inom rimlig tid, om inte lag kräver fortsatt lagring.

10. Tillämplig lag
Dessa villkor regleras av svensk lag. Tvister ska avgöras av svensk domstol.
